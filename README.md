# CF Lens 🔍

**CF Lens** is a Codeforces analytics dashboard built with React and Vite that helps competitive programmers gain deeper insights into their coding journey through interactive visualizations, performance analysis, and user comparisons.

🌐 **Live Demo:** <https://mayank171006.github.io/cf-lens/>

⭐ Explore analytics, comparisons, and problem-solving insights directly in the browser.

---

## Features

### 📊 Analytics Dashboard

Analyze your Codeforces journey through multiple visualizations:

- Rating vs Time
- Rank vs Time
- Contest Performance Analysis
- Contest Rank Distribution
- Consistency Metrics

### ⚔️ User Comparison

Compare two Codeforces users across various performance indicators:

- Rating Comparison
- Contest Performance Comparison
- Problem Rating Distribution
- Tag Strength Comparison
- One-Shot Solve Statistics
- Average Attempts per Solved Problem
- Average Solved Problem Rating
- Median Solved Problem Rating
- Unique Tags Solved

### 🧩 Problem Analysis

Understand your solving patterns and strengths:

- Problem Difficulty Distribution
- Rating-Based Analysis
- Tag-Based Analysis
- Solving Trends

### 🎨 User Experience

- Responsive Interface
- Interactive Charts
- Real-time Data Fetching from the Codeforces API
- Clean and Simple UI
- Loading and Error Handling

## Run Locally

Clone the repository:

```bash
git clone "https://github.com/Mayank171006/cf-lens.git"
```

Navigate to the project directory:

```bash
cd cf-lens
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```

## Motivation

Codeforces provides a large amount of valuable competitive programming data, but extracting meaningful insights from it can be difficult.

CF Lens was built to make it easier to:

- Track long-term progress
- Identify strengths and weaknesses
- Analyze problem-solving behavior
- Compare performance with other users
- Visualize coding growth over time
